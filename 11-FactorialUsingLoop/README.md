# Factorial Calculator (Using Loop in Java)

## Description
This is a simple Java program that calculates the **factorial** of a given non-negative integer using an iterative loop.  
The program uses a `for` loop to multiply consecutive integers from 1 up to the entered number and outputs the result.

---

## Features
- Accepts integer input from the user
- Calculates factorial using **iteration** (loop inside `main`)
- Handles large numbers using `long` data type
- Prints the factorial in a clean format
- Properly closes `Scanner` to free resources

---

## How to Use
1. Compile the program:

javac FactorialLoop.java
