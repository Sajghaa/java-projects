# Greatest of Three Numbers (Using Nested If Statements in Java)

## Description
This Java program finds the **greatest of three numbers** entered by the user using **nested if statements**.  
It demonstrates the use of conditional logic to compare multiple numbers and determine the largest one.

---

## Features
- Accepts three integer inputs from the user
- Uses **nested if statements** for comparison
- Prints the greatest number in a clear format
- Handles any integer input, including negative numbers
- Properly closes the `Scanner` object

---

## How to Use
1. Compile the program:
```bash
javac GreatestOfThree.java
